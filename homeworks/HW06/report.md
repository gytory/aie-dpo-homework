## 2. Protocol

- Разбиение: train/test 0.25/0.75, randomstate - 42
- Подбор: CV на train подбирался с помощью GridSearch, чтобы найти наилучший результат модели и избежать переобучения в случае деревьев.
- Метрики: accuracy, F1, ROC-AUC. Метрика accuracy уместна практически всегда, так как дает обобщенный результат по точности предсказаний. F1 позволяет более детально рассмотреть результат и сделать выводы по нему, а ROC-AUC более точная для моделей, выдающих вероятности определения класса.

## 5. Analysis

- Устойчивость: Random Forest достаточно устойчив к перемешиваюнию, потому что состоит из множества моделей и выбирает лучшую.
Decision Tree неустойчива, тк состоит из одной модели, склонной к подстраиванию к выборке, поэтому результат меняется достаточно сильно.
